# c2

Одностраничный Winamp-style калькулятор с метрономом.

## Возможности
- Вычисляет базовые операции `+`, `-`, `×`, `÷`, процент и смену знака.
- Интуитивные кнопки с подсветкой; `AC` выделена красным.
- Встроенный метроном на Web Audio с диапазоном 40–200 BPM и плавным бегунком темпа.

## Быстрый старт
1. Открой `winamp-calculator.html` в браузере.
2. Пользуйся калькулятором; метроном включается кнопкой `Старт`.
